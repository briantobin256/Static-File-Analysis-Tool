Description:

The first step involved in file analysis is to backup the file. A hash will also be used for verification.


Hashing

This tool generates a SHA-1 hash of the raw data of a file. This value will be completely different if any of the files data is altered even slightly.


Backup

A copy of a file is created at a chosen location, then a hash of this new file is generated and compared with the original to guarantee the backup is the same.


Your Goal:

To click the create backup button and select a location for backups to be stored. Also to be aware that backups should always be created before performing any analysis.
