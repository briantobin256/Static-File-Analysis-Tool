Static File Analysis Tool

This tool is designed as an introduction to static file analysis.
There is a checklist that you can follow which can be accessed
by clicking "Checklist" then "Show / Hide".
This gives a description of of each of the analysis techniques this tool provides.
There is also a notepad in each section for you to take notes.


To start, click "File" then "Open".
Select the file you want to analyse and follow the checklist.
